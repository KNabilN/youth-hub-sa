

# إصلاح تصدير CSV للمستخدمين

## المشكلة
تصدير CSV يعرض العناوين فقط بدون بيانات. السبب: استعلام التصدير يستخدم `user_roles(role)` كـ join مباشر قد يفشل بصمت، بينما الـ hook الأساسي `useAdminUsers` يستخدم استعلامين منفصلين وينجح.

## الحل
تعديل كود التصدير في `src/components/admin/UserTable.tsx` ليستخدم نفس نمط الاستعلامين المنفصلين:

1. استعلام أول: جلب كل الـ profiles بدون حد صفحات
2. استعلام ثاني: جلب كل الـ user_roles
3. دمجهما يدوياً عبر Map ثم تصدير CSV

## الملف المتأثر
- `src/components/admin/UserTable.tsx` — تعديل onClick handler لزر التصدير فقط
