

# إصلاح تصدير تقرير PDF

## المشكلة
تصدير PDF يفشل بسبب مشاكل في معالجة النصوص العربية مع `@react-pdf/renderer`. الخط لا يتطابق مع صفحة التقرير.

## الحل
استبدال طريقة التصدير الحالية (`@react-pdf/renderer`) بنهج `html2canvas` + `jsPDF` الذي يلتقط صفحة التقرير كما تظهر بالضبط ويحولها لـ PDF — مما يضمن تطابق الخط والتنسيق.

## التغييرات

### 1. إضافة مكتبات جديدة
- `jspdf`
- `html2canvas`

### 2. تعديل `src/lib/report-pdf.ts`
- استبدال `generateReportPDF` بدالة جديدة تستخدم `html2canvas` لالتقاط عنصر DOM محدد وتحويله لـ PDF عبر `jsPDF`
- تدعم تقسيم الصفحات تلقائياً (multi-page) لو المحتوى طويل

### 3. تعديل `src/pages/admin/AdminReports.tsx`
- إضافة `ref` لمنطقة محتوى التقرير (الإحصائيات + الرسوم البيانية + الجداول)
- تعديل `exportPDF` ليلتقط المحتوى المرئي مباشرة بدلاً من بناء مستند PDF منفصل
- إخفاء أزرار التصدير والفلاتر أثناء الالتقاط

### 4. لا حذف لملفات PDF الحالية
`ReportDocument.tsx` و `pdf-fonts.ts` و `pdf-utils.ts` تبقى كما هي لأنها مستخدمة في الفواتير أيضاً.

## النتيجة
- PDF يطابق شكل الصفحة تماماً (نفس الخط، الألوان، الرسوم البيانية)
- دعم كامل للعربية بدون مشاكل
- تقسيم تلقائي للصفحات
