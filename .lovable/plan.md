
# إعادة بناء الصفحة الرئيسية لتطابق الموقع القديم

## الهدف
نقل محتوى وتصميم الموقع القديم (sharedservices.solutions) إلى الصفحة الرئيسية الحالية، مع إضافة صفحات "من نحن" و"الأسئلة الشائعة" كصفحات مستقلة، وتحديث الهيدر والفوتر ليتوافقا مع القائمة العلوية للموقع القديم.

## التغييرات المطلوبة

### 1. تحديث الهيدر (Header) - إضافة قائمة تنقل
**الوضع الحالي:** الهيدر يحتوي فقط على الشعار + تسجيل الدخول + إنشاء حساب
**المطلوب:** إضافة قائمة تنقل تحتوي على:
- من نحن (`/about`)
- الأسئلة الشائعة (`/faq`)
- شروط الخدمة (`/privacy`)
- تسجيل الدخول / إنشاء حساب

### 2. تحديث قسم Hero
**المطلوب:** تحديث النصوص لتتوافق مع الموقع القديم:
- العنوان: "كل ماتحتاجه في مكان واحد"
- الوصف: "ازدهر في منظومة مزوّدي الخدمة، حيث التميّز والفرص بلا حدود"
- إضافة وصف فرعي: "منصتك الإلكترونية الشاملة لكل ما تحتاجه في مكان واحد"

### 3. إضافة قسم الإحصائيات من الموقع القديم
**المطلوب:** إضافة قسم إحصائيات بصري مع أيقونات (مشابه للموقع القديم):
- متوسط التقييم: 4.91/5
- 211K+ عقد
- 1,665 مهارة مدعومة

### 4. إضافة قسم "آراء العملاء" (Testimonials)
**المطلوب:** قسم جديد يعرض آراء العملاء من الموقع القديم بنظام بطاقات:
- سعود المطيري - جمعية قمم
- فهد العتيبي - جمعية قمة
- نواف القحطاني - جمعية الود
- عبدالله الشهري - جمعية مداد
- نورة السبيعي - جمعية مزن

### 5. تحديث الفوتر ليتوافق مع الموقع القديم
**المطلوب:** فوتر أكثر تفصيلاً يحتوي على:
- الشعار + اسم المنصة + وصف مختصر
- الفئات الأعلى تقييماً (الخدمات المالية، البناء المؤسسي، العمليات الداخلية، إلخ)
- معلومات التواصل (البريد: sspf.scy@gmail.com، الهاتف: 0554648475)
- روابط: من نحن، الأسئلة الشائعة، سياسة الخصوصية

### 6. إنشاء صفحة "من نحن" (`/about`)
صفحة مستقلة تحتوي على محتوى الموقع القديم:
- **ما هي منصة الخدمات المشتركة؟** - وصف المنصة والمجالات (الإدارية، التقنية، المالية، القانونية، التدريبية)
- **لماذا منصة الخدمات المشتركة؟** - المزايا الأربعة
- **من هم المستهدفون؟** - الجمعيات، الشركات، الداعمون
- **ما هي الخدمات؟** - 6 فئات خدمات
- **ما المزايا؟** - 5 مزايا رئيسية
- **كيف تحقق أكبر فائدة؟** - 4 خطوات
- **كيف نختار مزودي الخدمات؟** - معايير الاختيار
- **طموحاتنا المستقبلية** - 4 أهداف
- **الجودة والأمان** - 4 ضمانات

### 7. إنشاء صفحة "الأسئلة الشائعة" (`/faq`)
صفحة مستقلة بنظام Accordion تحتوي على:
- **أولاً:** أسئلة الجمعيات الخيرية (10 أسئلة)
- **ثانياً:** أسئلة مزودي الخدمات (8 أسئلة)
- **ثالثاً:** أسئلة عامة وإدارية (8+ أسئلة)

### 8. إضافة CMS sections لصفحتي About و FAQ
إضافة `about` و `faq` كأقسام جديدة في جدول `site_content` ليتمكن المدير من تعديلها لاحقاً، وإضافتها في CMS page groups.

## التفاصيل التقنية

### الملفات الجديدة:
1. **`src/pages/About.tsx`** - صفحة من نحن بتصميم أقسام متعاقبة مع خلفيات متبادلة
2. **`src/pages/FAQ.tsx`** - صفحة الأسئلة الشائعة بنظام Accordion مقسمة لـ 3 فئات

### الملفات المعدلة:
1. **`src/pages/Index.tsx`** - إضافة قسم Testimonials + تحديث الهيدر بقائمة تنقل + تحديث الفوتر
2. **`src/App.tsx`** - إضافة routes لـ `/about` و `/faq`
3. **`src/pages/admin/AdminCMS.tsx`** - إضافة "about" و "faq" في page groups

### Migration SQL:
```text
INSERT INTO site_content (section_key, content)
VALUES
  ('about', '{ JSON content for about page }'),
  ('faq', '{ JSON content for FAQ page }'),
  ('testimonials', '{ JSON content for testimonials }')
ON CONFLICT (section_key) DO NOTHING;
```
