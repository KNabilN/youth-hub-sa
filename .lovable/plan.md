

# إضافة دليل الاستخدام لكل دور في لوحة التحكم

## الفكرة
صفحة جديدة `/guide` تظهر في السايدبار تحت "الصفحة الرئيسية" مباشرة، تعرض دليل استخدام مخصص حسب دور المستخدم (جمعية شبابية / مزود خدمة / مانح) باستخدام Accordion لعرض شرح كل مهمة.

## التنفيذ

### 1. صفحة جديدة `src/pages/UserGuide.tsx`
- تستخدم `useAuth` لمعرفة الدور
- تعرض محتوى ثابت مقسم بـ Accordion حسب الدور:

**جمعية شبابية** (8 أقسام):
إنشاء طلب جديد، متابعة الطلبات، تصفح سوق الخدمات وشراء خدمة، إدارة العقود، طلبات المنح والمنح المستلمة، تقارير الأثر، الفواتير، الشكاوى والتقييمات

**مزود خدمة** (8 أقسام):
إضافة خدمة، تصفح طلبات الجمعيات وتقديم عرض، متابعة العروض، إدارة العقود، المعاملات المادية والسحب، الفواتير، الشكاوى والتقييمات، سوق الخدمات

**مانح** (5 أقسام):
تصفح الجمعيات، طلبات الدعم الواردة، تقديم منحة، الفواتير، تقارير الأثر

### 2. تعديل `AppSidebar.tsx`
- إضافة عنصر `{ title: "دليل الاستخدام", url: "/guide", icon: BookOpen }` بعد "الصفحة الرئيسية" لكل الأدوار الثلاثة

### 3. تعديل `App.tsx`
- إضافة route محمي: `/guide` → `UserGuide`

### الملفات المتأثرة:
- `src/pages/UserGuide.tsx` — جديد
- `src/components/AppSidebar.tsx` — إضافة رابط دليل الاستخدام
- `src/App.tsx` — إضافة الراوت
