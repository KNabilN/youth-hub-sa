

# إصلاح عرض الخدمات المتوفرة في الصفحة الرئيسية

## المشكلة
قسم "الخدمات المتوفرة" لا يظهر لأن سياسة الأمان (RLS) على جدول `micro_services` تسمح فقط للمستخدمين المسجلين بعرض الخدمات المعتمدة. الزائر غير المسجل لا يستطيع رؤية أي خدمة.

## الحل
إضافة سياسة أمان جديدة تسمح للزائرين غير المسجلين بعرض الخدمات المعتمدة فقط.

## التغييرات

### 1. إضافة migration جديدة
إنشاء سياسة RLS جديدة على جدول `micro_services`:

```sql
CREATE POLICY "Public browse approved services"
  ON public.micro_services
  FOR SELECT
  TO anon
  USING (approval = 'approved');
```

هذا يسمح لزوار الصفحة الرئيسية (غير المسجلين) برؤية الخدمات المعتمدة فقط.

### 2. لا تغيير في الكود
الكود الحالي في `Index.tsx` يعرض القسمين بالترتيب الصحيح بالفعل:
1. طلبات الجمعيات (LandingRequestsTable)
2. الخدمات المتوفرة (LandingServicesGrid)

المشكلة فقط في صلاحيات قاعدة البيانات.
