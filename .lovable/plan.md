

# تحميل جميع الخدمات في نفس الصفحة بدلاً من التنقل

## المشكلة
زر "تصفح جميع الخدمات" ينقل المستخدم إلى `/marketplace` أو `/auth`، بينما المطلوب تحميل باقي الخدمات في نفس الصفحة.

## الحل
تحويل الزر من رابط تنقل إلى زر "تحميل المزيد" يجلب كل الخدمات المعتمدة ويعرضها في نفس المكان.

## التغييرات

### 1. `src/components/landing/LandingServicesGrid.tsx`
- إضافة state داخلي `showAll` (افتراضي `false`)
- عند الضغط على الزر، يتم استدعاء query جديد لجلب كل الخدمات المعتمدة (بدون limit)
- عرض الخدمات الإضافية في نفس الشبكة
- تغيير نص الزر بعد التحميل إلى "إخفاء" أو إخفاء الزر بالكامل
- إضافة prop `onLoadAll` callback أو تنفيذ الاستعلام داخلياً

### التنفيذ التقني:
```
- state: showAll, allServices[], allLoading
- عند النقر: استعلام supabase بدون limit → تحديث القائمة المعروضة
- عرض: showAll ? allServices : services (أول 9)
- الزر يتحول لـ "عرض أقل" عند التوسيع
```
