
# إكمال الملف الشخصي الإلزامي + بيانات الحساب البنكي

## ملخص
بعد تسجيل الدخول، يظهر شريط تنبيه علوي (Banner) يطلب من المستخدم إكمال بياناته الإلزامية حسب دوره. يمكنه التنقل بحرية لكن الشريط يبقى ظاهراً حتى يكمل جميع الحقول المطلوبة. بالإضافة لذلك، يتم إضافة بيانات الحساب البنكي كحقول إلزامية لمقدمي الخدمة والجمعيات.

---

## 1. تغييرات قاعدة البيانات

إضافة أعمدة جديدة لجدول `profiles`:

```sql
ALTER TABLE profiles
  ADD COLUMN bank_name text DEFAULT '',
  ADD COLUMN bank_account_number text DEFAULT '',
  ADD COLUMN bank_iban text DEFAULT '',
  ADD COLUMN bank_account_holder text DEFAULT '';
```

لا حاجة لسياسات RLS إضافية - السياسات الحالية تغطي القراءة والتعديل.

---

## 2. دالة فحص اكتمال الملف الشخصي (Hook جديد)

إنشاء `src/hooks/useProfileCompleteness.ts`:

منطق الفحص حسب الدور:

**جميع الأدوار (مشترك):**
- `full_name` (مطلوب)
- `phone` (مطلوب)

**الجمعيات (youth_association) - إضافي:**
- `organization_name` (مطلوب)
- `license_number` (مطلوب)
- `contact_officer_name` (مطلوب)
- `contact_officer_phone` (مطلوب)
- `bank_name` (مطلوب)
- `bank_account_number` (مطلوب)
- `bank_iban` (مطلوب)
- `bank_account_holder` (مطلوب)

**مقدمو الخدمة (service_provider) - إضافي:**
- `bio` (مطلوب)
- `hourly_rate` (مطلوب)
- `bank_name` (مطلوب)
- `bank_account_number` (مطلوب)
- `bank_iban` (مطلوب)
- `bank_account_holder` (مطلوب)

**المانحون (donor) - إضافي:**
- `bio` (اختياري - لا حقول إضافية إلزامية)

الدالة تُرجع:
- `isComplete: boolean` - هل الملف مكتمل؟
- `missingFields: string[]` - قائمة الحقول الناقصة بالعربي
- `completionPercentage: number` - نسبة الإكمال

## 3. شريط التنبيه العلوي (Profile Completion Banner)

إنشاء `src/components/ProfileCompletionBanner.tsx`:

- شريط أعلى المحتوى (أسفل الهيدر مباشرة) بلون `warning` خفيف
- يعرض نسبة الإكمال مع شريط تقدم (Progress bar)
- رسالة واضحة: "يرجى إكمال بيانات ملفك الشخصي لتفعيل جميع ميزات المنصة"
- زر "إكمال الملف الشخصي" يوجّه لصفحة `/profile`
- يظهر فقط عند `isComplete === false`
- يمكن إخفاؤه مؤقتاً (dismiss) لكنه يعود عند تحميل الصفحة مجدداً
- يتبع نمط التصميم الموحد مع أيقونة ملونة وتأثير fade-in

**موقع الإدراج:** داخل `DashboardLayout.tsx` قبل `{children}` مباشرة.

## 4. تعديل صفحة الملف الشخصي (Profile.tsx)

### إضافة قسم البيانات البنكية:
- بطاقة جديدة "البيانات البنكية" تظهر لمقدمي الخدمة والجمعيات فقط
- أيقونة `Landmark` (بنك) بلون primary
- حقول: اسم البنك، رقم الحساب، رقم IBAN، اسم صاحب الحساب
- حقل IBAN يبدأ بـ `SA` مع تحقق من الطول (24 حرف)

### تمييز الحقول الإلزامية:
- إضافة نجمة حمراء `*` بجانب كل حقل إلزامي
- عرض رسالة تحقق عند محاولة الحفظ بحقول ناقصة
- تلوين الحقول الفارغة الإلزامية بحدود تحذيرية

### عرض نسبة الإكمال:
- شريط تقدم في أعلى صفحة الملف الشخصي يوضح نسبة الإكمال
- قائمة بالحقول المتبقية مع علامات اختيار للمكتملة
