
# اضافة صفحة تفاصيل التذكرة للمدير

## الهدف
إنشاء صفحة تفاصيل شاملة لتذاكر الدعم الفني في لوحة الإدارة، مع اتباع نفس أنماط التصميم المستخدمة في صفحة تفاصيل المستخدم (AdminUserDetail).

## التصميم المقترح

الصفحة ستتضمن:

1. **شريط علوي ثابت** مع زر العودة وزر تغيير الحالة
2. **قسم رئيسي (Hero)** يعرض موضوع التذكرة مع شارات الحالة والأولوية واسم المستخدم وتاريخ الإنشاء
3. **بطاقات معلومات** بأيقونات توضيحية (الموضوع، المستخدم، الأولوية، الحالة، التاريخ)
4. **قسم الوصف** لعرض تفاصيل التذكرة كاملة
5. **إمكانية تغيير الحالة** مباشرة من صفحة التفاصيل

## الملفات المطلوبة

### 1. صفحة التفاصيل الجديدة: `src/pages/admin/AdminTicketDetail.tsx`
- تستخدم `useParams` لقراءة معرف التذكرة من الرابط
- تجلب بيانات التذكرة مع بيانات المستخدم (profiles)
- تعرض Hero section مشابه لـ AdminUserDetail
- تعرض بطاقات InfoField للمعلومات الأساسية
- تتيح تغيير الحالة مع إشعار نجاح/خطأ
- حالة التحميل وحالة عدم العثور على التذكرة

### 2. Hook جديد: `src/hooks/useAdminTicketById.ts`
- يجلب تذكرة واحدة بالمعرف مع بيانات المستخدم المرتبط

### 3. تحديث `src/pages/admin/AdminTickets.tsx`
- جعل صف التذكرة قابل للنقر للانتقال لصفحة التفاصيل
- إضافة مؤشر النقر (cursor-pointer)

### 4. تحديث `src/App.tsx`
- إضافة مسار `/admin/tickets/:id` للصفحة الجديدة

## التفاصيل التقنية

### هيكل صفحة التفاصيل:
```text
+------------------------------------------+
|  [تغيير الحالة]        [العودة للتذاكر ->] |  <- شريط ثابت
+------------------------------------------+
|                                          |
|   [أيقونة تذكرة]                         |
|   موضوع التذكرة                          |
|   [شارة الحالة] [شارة الأولوية]          |  <- Hero Section
|   اسم المستخدم                           |
|   تاريخ الإنشاء                          |
|                                          |
+------------------------------------------+
|                                          |
|  [المستخدم]  [الأولوية]  [الحالة]        |
|  [التاريخ]   [آخر تحديث]                 |  <- بطاقات InfoField
|                                          |
+------------------------------------------+
|                                          |
|  وصف التذكرة                             |  <- بطاقة الوصف
|  النص الكامل للتذكرة...                  |
|                                          |
+------------------------------------------+
```

### Hook البيانات:
```typescript
// useAdminTicketById.ts
useQuery({
  queryKey: ["admin-ticket", id],
  queryFn: async () => {
    const { data } = await supabase
      .from("support_tickets")
      .select("*, profiles:user_id(full_name, avatar_url)")
      .eq("id", id)
      .single();
    return data;
  }
});
```

### المسار الجديد:
```
/admin/tickets/:id -> AdminTicketDetail
```

### تحديث جدول التذاكر:
- النقر على أي صف ينتقل إلى `/admin/tickets/{id}`
- إضافة `onClick` و `cursor-pointer` على `TableRow`
