
# تغيير "النزاعات" إلى "الشكاوى" في كل النظام

## الملخص
تحديث جميع النصوص العربية لاستبدال "النزاعات/نزاع/متنازع" بـ "الشكاوى/شكوى" في كل الصفحات والمكونات.

- "النزاعات" ← "الشكاوى" (قائمة الجمعية/مقدم الخدمة + المدير)

- "رفع النزاع" ← "رفع الشكوى"

- "حدث خطأ أثناء رفع النزاع" ← "حدث خطأ أثناء رفع الشكوى"
- "لا توجد نزاعات" ← "لا توجد شكاوى"
