

# تحسين تصدير CSV مع نافذة اختيار

## الفكرة
استبدال زر التصدير المباشر بنافذة Dialog تتيح للمستخدم:
1. اختيار الدور (الكل / جمعية شبابية / مقدم خدمة / مانح / مدير النظام)
2. اختيار الأعمدة المطلوبة (checkboxes) مثل: الاسم، الهاتف، المنظمة، الدور، التوثيق، الحالة، تاريخ الانضمام
3. زر تصدير داخل النافذة

## التغييرات

### `src/components/admin/UserTable.tsx`
- إضافة state للـ export dialog: `exportOpen`, `exportRole`, `exportColumns`
- استبدال onClick المباشر بفتح الـ dialog
- إنشاء Dialog يحتوي:
  - Select لاختيار الدور (فلتر التصدير)
  - مجموعة Checkboxes لاختيار الأعمدة
  - زر "تصدير" ينفذ الاستعلام مع تطبيق فلتر الدور عبر filter على النتائج بعد الجلب
- عند التصدير: جلب البيانات → فلترة بالدور المختار → تصدير الأعمدة المحددة فقط

## النتيجة
نافذة منظمة تمكّن المدير من تخصيص التصدير حسب الدور والأعمدة المطلوبة.
