# استبدال قسم CTA الحالي بتصميم مطابق للصور المرجعية وإضافة قسم اشتراك بريدي

## التغييرات المطلوبة

### تحديث `src/pages/Index.tsx`

**1. قسم CTA الحالي (القسم 8) - يبقى كما هو** بتصميمه الداكن مع الحدود المتوهجة (مطابق للصورة الثانية).

**2. إضافة قسم اشتراك بريدي جديد** بين قسم الشهادات وقسم CTA الداكن، بتصميم مطابق للصورة الأولى:

- خلفية زرقاء صلبة (`bg-[#2381C0]` أو قريب منها)
- عنوان أبيض عريض: "ابدأ رحلتك المهنية اليوم"
- وصف أبيض شفاف: "اشترك في قائمتنا البريدية ليصلك كل جديد"
- حقل إدخال بريد إلكتروني أبيض مع placeholder "أدخل بريدك الإلكتروني" -- لا يدخل المستخدم بريد الكتروني
- زر "ابدأ الآن" أبيض مع سهم
- الحقل والزر بجانب بعض أفقياً (flex row)

### التفاصيل التقنية

```text
+--------------------------------------------------+
|          خلفية زرقاء (bg-[#2381C0])               |
|                                                    |
|         ابدأ رحلتك المهنية اليوم                  |
|    اشترك في قائمتنا البريدية ليصلك كل جديد       |
|                                                    |
|   [ ابدأ الآن  ← ]  [ أدخل بريدك الإلكتروني    ] |
|                                                    |
+--------------------------------------------------+
```

- القسم الجديد يُضاف كقسم مستقل في `Index.tsx` بعد Testimonials وقبل CTA الداكن
- حقل الإدخال والزر يستخدمان `flex-row-reverse` أو ترتيب RTL طبيعي
- على الجوال: الحقل والزر يتحولان لعمود (flex-col)
- الزر يستخدم `bg-white text-[#2381C0]` مع hover effect
