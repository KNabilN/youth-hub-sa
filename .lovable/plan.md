
# إضافة صفحة تفاصيل الخدمة للأدمن

## الهدف
السماح للأدمن بالنقر على عنوان الخدمة في جدول إدارة الخدمات لفتح صفحة تفاصيل كاملة تعرض جميع معلومات الخدمة مع إمكانية تغيير الحالة والتعديل المباشر.

## التغييرات المطلوبة

### 1. إنشاء صفحة `src/pages/admin/AdminServiceDetail.tsx`
صفحة جديدة داخل `DashboardLayout` تعرض:
- صورة الخدمة والمعرض
- العنوان والوصف الكامل والوصف التفصيلي
- معلومات مقدم الخدمة (الاسم، الصورة)
- التصنيف والمنطقة
- السعر والباقات
- الأسئلة الشائعة
- عدد المشاهدات والمبيعات
- حالة الموافقة مع إمكانية تغييرها مباشرة
- زر التعديل المباشر
- زر العودة للقائمة

### 2. تعديل `src/App.tsx`
- إضافة استيراد lazy للصفحة الجديدة
- إضافة مسار `/admin/services/:id` داخل `AdminRoute`

### 3. تعديل `src/pages/admin/AdminServices.tsx`
- جعل عنوان الخدمة رابطاً قابلاً للنقر ينتقل إلى `/admin/services/{id}`
- إضافة زر "عرض" بجانب زر "تعديل" في عمود الإجراءات

### التفاصيل التقنية
- الصفحة الجديدة تستخدم `useServiceDetail` الموجود لجلب البيانات الكاملة
- تستخدم `useUpdateServiceApproval` و `useAdminUpdateService` للتعديل
- تعيد استخدام مكونات `ServiceGallery` و `ServicePackages` و `ServiceFAQ` الموجودة
- تُغلف داخل `DashboardLayout` للحفاظ على تصميم لوحة الأدمن
