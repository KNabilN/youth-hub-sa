

## إضافة جميع حقول الملف الشخصي إلى نموذج إنشاء مستخدم جديد

### الملخص
تحديث نموذج إنشاء المستخدم في لوحة الأدمن ليشمل جميع الحقول الظاهرة في الصورة المرجعية، بما يتطابق مع حقول جدول `profiles`.

### الحقول المطلوب إضافتها
الحقول الموجودة حالياً: البريد الإلكتروني، كلمة المرور، الاسم الكامل، الدور، رقم الهاتف، اسم المنظمة.

الحقول الجديدة:
- **رقم الترخيص** (license_number) - نص
- **اسم ضابط الاتصال** (contact_officer_name) - نص
- **رقم ضابط الاتصال** (contact_officer_phone) - نص
- **بريد ضابط الاتصال** (contact_officer_email) - بريد إلكتروني
- **صفة ضابط الاتصال** (contact_officer_title) - نص
- **نبذة** (bio) - textarea
- **السعر بالساعة** (hourly_rate) - رقم

### التغييرات

**تعديل `src/components/admin/AdminCreateUserDialog.tsx`**
- إضافة state variables لكل حقل جديد
- تحسين حقل الهاتف ليشمل رمز الدولة +966 مثل نموذج التسجيل العادي
- إضافة حقول الإدخال الجديدة في الفورم
- تحديث `handleSubmit` لإرسال جميع الحقول الجديدة عند تحديث الـ profile
- إضافة `ScrollArea` لأن الفورم سيكون طويلاً
- تحديث `resetForm` لتشمل جميع الحقول
- توسيع عرض الـ Dialog إلى `sm:max-w-lg`

### تفاصيل تقنية

لا حاجة لتغييرات في قاعدة البيانات -- جميع الحقول موجودة بالفعل في جدول `profiles`.
